## 템플릿 정보

### react-router-cloudflare (SSR)

| 항목 | 값 |
|------|---|
| 프레임워크 | React Router 7 |
| 렌더링 | SSR (Server-Side) |
| 배포 | Cloudflare Workers |
| 스타일링 | Tailwind CSS 4 |
| 빌드 | Vite 7 |

### react-router-spa (SPA)

| 항목 | 값 |

| 렌더링 | CSR (Client-Side) |
| 배포 | Cloudflare Pages (정적) |
| 스타일링 | Tailwind CSS 4 |
| 빌드 | Vite 7 |
