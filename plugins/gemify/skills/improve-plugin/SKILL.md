# Improve Plugin Skill

플러그인 경로를 워크스페이스에 추가하고 **개선 작업을 수행**합니다.

## 동작

1. 플러그인 경로 확인
2. `/add-dir {경로}` 실행 제안 (사용자 승인 필요)
3. inbox/materials에서 관련 개선 아이디어 검색
4. 플러그인 구조 파악
5. 개선 작업 수행

## 워크플로우

```
inbox/materials에 플러그인 개선 아이디어
    ↓
/add-dir {플러그인 경로}
    ↓
개선 작업 수행
    ↓
세션 종료 → 자동 스코프 해제
```

## 플러그인 구조 파악

```
{plugin-path}/
├── .claude-plugin/
│   └── plugin.json      # 플러그인 메타데이터
├── commands/            # 슬래시 커맨드
├── skills/              # 스킬 정의
└── assets/              # 에셋 파일
```

## 개선 아이디어 검색

inbox/materials에서 플러그인 관련 파일 검색:
- 파일명에 플러그인 이름 포함
- 내용에 플러그인 개선 관련 키워드

## 규칙

- add-dir은 **사용자 승인 후 실행**
- 플러그인 구조 파악 후 작업 시작
- 세션 종료 시 자동으로 스코프 해제됨 (remove-dir 불필요)
- 작업 완료 후 변경사항 요약

## add-dir 제안 예시

```
플러그인 개선을 위해 해당 경로를 워크스페이스에 추가할까요?

/add-dir ~/plugins/my-plugin

(세션 종료 시 자동으로 스코프에서 해제됩니다)
```
