# Zettelkasten 핵심 원리

## 1. Atomic Notes
- 하나의 메모에 하나의 아이디어만
- 완결성 있는 단위

## 2. Links
- 메모 간 연결이 핵심
- 폴더 분류보다 연결이 중요

## 3. Permanent Notes
- 임시 메모 → 영구 메모로 발전
- 자신의 말로 재작성

## 내 생각
이 방식을 gemify 파이프라인에 적용하면:
- inbox = 임시 메모
- drafts = 발전 단계
- library = 영구 메모
