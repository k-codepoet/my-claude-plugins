# /forgeify:update

update 스킬을 사용하여 플러그인을 최신 가이드라인으로 갱신합니다.

## 사용법

```
/forgeify:update                       # 현재 디렉토리 플러그인 업데이트
/forgeify:update ./plugins/my-plugin   # 특정 플러그인 업데이트
/forgeify:update --check-only          # 변경 필요 사항만 확인 (수정 없음)
```
