# Namify 도움말

> Name it right.

제품/서비스 네이밍을 도와주는 도구입니다. 메타포 탐색 → 이름 생성 → 검증 → 추천 흐름으로 진행합니다.

## 사용 가능한 커맨드

### 가이드
| 커맨드 | 설명 |
|--------|------|
| `/namify:help` | 이 도움말 표시 |
| `/namify:howto [topic]` | 사용 가이드 (주제별 사용법과 예시) |

### 네이밍
| 커맨드 | 설명 |
|--------|------|
| `/namify:name` | 제품/서비스 이름 생성 |

## 네이밍 흐름

```
1. 핵심 가치 추출
       ↓
2. 메타포 탐색 (광물, 식물, 건축, 도구, 과학)
       ↓
3. 이름 후보 생성 (-ify, Craft, Lab, Hub 등)
       ↓
4. 검증 (발음, 기존 의미, 검색 가능성, 문화적 적합성)
       ↓
5. 추천 + 이유
```
