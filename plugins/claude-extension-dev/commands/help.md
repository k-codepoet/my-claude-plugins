# Claude Extension Dev 도움말

Claude Code 확장(플러그인, 스킬, 에이전트 등) 개발을 위한 한국어 가이드입니다.

## 포함된 스킬

이 플러그인은 Claude가 자동으로 활성화하는 7개의 스킬을 제공합니다:

| 스킬 | 설명 |
|------|------|
| `plugin-guide` | 플러그인 구조, plugin.json 작성법 |
| `command-guide` | 슬래시 커맨드 작성법 |
| `agent-guide` | 서브에이전트 정의 및 활용법 |
| `skill-guide` | SKILL.md 작성법, Agent Skills 표준 |
| `hook-guide` | 이벤트 기반 훅 작성법 |
| `marketplace-guide` | 마켓플레이스 구축 및 배포 |
| `workflow-guide` | 실전 개발 워크플로우 예시 |

## 사용 방법

스킬은 Claude가 대화 컨텍스트에 따라 자동으로 활성화합니다.

**예시 질문:**
- "플러그인 만드는 방법 알려줘"
- "agent 파일 어떻게 작성해?"
- "hook 이벤트 종류가 뭐야?"
- "marketplace.json 스키마 알려줘"
