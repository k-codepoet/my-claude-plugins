# 제품 네이밍 가이드

> "Name it right" - 좋은 이름은 기억에 남고, 제품이 뭘 하는지 암시하며, 시리즈로 확장 가능하다.

## 핵심 원칙

**브랜드는 기억에 남게, UX는 생각 없이 쓸 수 있게.**

```
브랜드 레이어: "Turn your thoughts into gems" (감성, 마케팅)
        ↑ 분리 ↑
UX 레이어: inbox/ → drafts/ → library/ (직관, 기능)
```

## 이름 생성 흐름

```
핵심 가치 → 메타포 탐색 → 패턴 적용 → 검증 → 선택
```

## 주요 패턴

| 패턴 | 의미 | 예시 |
|------|------|------|
| -ify | 변환/생성 | Shopify, Gemify, Terrafy |
| Craft | 장인정신 | Minecraft, Craftify |
| Lab/Hub | 중심/연결 | GitLab, GitHub |
