# Example Subject View

## 구조

```
[생각] → [포착] → [정리]
   ↓        ↓        ↓
 떠오름   inbox    library
```

## 스토리

지식 관리의 첫 번째 원칙을 정리하기 위해 시작했다.

"일단 캡처"라는 핵심 원칙을 도출했고, 이를 기반으로 Gemify 파이프라인을 설계했다.

현재는 inbox → drafts → library 흐름이 완성되었고, views 레이어를 통해 지식을 주제별로 조합할 수 있게 되었다.
